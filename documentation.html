Features

Everything You Need to Sell Online

Enterprise comes packed with production-ready features for your e-commerce business.

Product Management

Full CRUD for products, categories (3-level), brands, attributes with values, colors, and product variants with SKU combinations.

Cart & Checkout

Guest and user cart, address management, shipping cost calculation, coupon application, and multi-payment checkout flow.

Order Management

Order listing, status tracking, payment updates, delivery assignment, bulk operations, and PDF invoice download.

Flash Deals & Coupons

Time-limited flash deals with multi-type coupon engine supporting conditional discounts and promotional campaigns.

Roles & Permissions

Granular RBAC with Admin, Staff, and Customer roles featuring fine-grained permission controls across the platform.

Analytics & Reports

Stock reports, sales reports, wish reports, user search analytics, wallet transaction history, and commission tracking dashboards.

Marketing & Comms

Newsletter management, bulk email campaigns, SMS via Twilio/Nexmo/SSL, Firebase push notifications, and Google reCAPTCHA.

Delivery Management

Delivery boy system with order assignment, collection and earnings summary, plus Steadfast Courier integration for tracking.

CMS & SEO

Blog management, custom pages (About, Terms, Privacy), banners, dynamic sitemap.xml, robots.txt, and per-page OG meta tags.

Wishlist & Reviews

Public and private wishlists, product reviews and ratings with admin moderation for quality control.

System Administration

General settings, SMTP config, currency/language management, file manager (Aiz Uploader), cache management, and database backup.

User Management

Email/password auth, social login (Google, Facebook), email verification, password reset, profile management, and address book.

Security

Enterprise-Grade Security Posture

Defense-in-depth approach protecting your platform across every layer — from network to application.

Authentication & Authorization

Multi-guard authentication (web + Sanctum API tokens), role-based middleware (IsAdmin, IsUser), email verification, password reset with throttling, social login OAuth 2.0.

SanctumRBACOAuth 2.0Email VerifyThrottle
CSRF & XSS Protection

Laravel VerifyCsrfToken on all web routes, Blade auto-escaping ({{ }}), React's built-in XSS prevention, custom htmlToPlainText sanitizer, JSON-only API responses.

CSRF TokensAuto-EscapingSanitization
SQL Injection Prevention

All queries go through Eloquent ORM with parameterized bindings. Zero raw SQL queries in controllers or models — completely immune to SQL injection attacks.

Eloquent ORMParameterizedNo Raw SQL
HTTPS & Secure Comms

HttpsProtocol middleware redirects HTTP to HTTPS (FORCE_HTTPS=On), Apache rewrite rules enforce SSL, production debug disabled (APP_DEBUG=false), sensitive env vars blacklisted from output.

HTTPS EnforceDebug Blacklist.htaccess
Session & Cookie Security

HttpOnly cookies, separate admin/frontend session cookies preventing session conflict, 24-hour session lifetime, configurable domain and path.

HttpOnlySession IsolationSecure Cookies
CORS & API Security

Custom CORS middleware with dynamic origin enforcement, Laravel CORS config, API rate limiting (60 requests/min), Sanctum token-based API auth with Bearer headers.

CORSRate LimitingBearer Auth
Input Validation & Uploads

Laravel Request validation (validate() + Form Requests), file type restrictions in Aiz Uploader, safe image manipulation via Intervention Image.

ValidationFile RestrictSafe Images
Encryption & Secrets

bcrypt password hashing (Laravel default), AES-256-CBC APP_KEY encryption, data encryption for sensitive lookups, all credentials stored in .env (excluded from version control).

bcryptAES-256-CBC.env Secrets
Additional Protections

Google reCAPTCHA (v2 Invisible), robots.txt disallowing sensitive paths, custom 404/500 error pages, maintenance mode, GitHub Actions with secrets-based deployment.

reCAPTCHARobots.txtError PagesCI/CD Secrets
